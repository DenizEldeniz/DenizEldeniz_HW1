My Projects

Thanks to my passion for software development, I have worked on many projects and continue to do so. Here are the projects I've put effort into and am proud of:

Lapiors - Photo Editor

Lapiors is a Canva-like photo editing platform. Users can edit their photos, apply filters, and create designs directly in the browser. I am developing this project entirely on my own — I handle all the planning, design, and coding from scratch.

I actually have a business idea in mind, and I started developing Lapiors to serve as the foundation for that idea. The project is still in development and I'm adding new features every day. Running such a comprehensive project on my own has taught me a lot in both technical and business development aspects. I plan to launch this project for users in the future.

Exagate Issue Tracking System

This project is an issue tracking system that I developed during my summer internship at Exagate company in July 2025. I completed the project from scratch within 20 business days during my internship. I worked closely with the technical team at the company to identify requirements and designed the system accordingly.

What I'm most proud of is that the project continues to be used permanently at the company even after my internship ended. It didn't just remain as an internship project — it is actively used in a real production environment. My internship mentor loved the project and they integrated it into the company's daily operations. This experience taught me what real-world software development truly means.

URL Shortener

I developed a URL shortening service that converts long URLs into short, shareable links. I used React on the frontend, .NET Web API on the backend, and MongoDB as the database. Being a full-stack project, it gave me the opportunity to improve both my frontend and backend skills.

The project has not been deployed yet and only runs locally. But I'm planning to get a domain and deploy it in the future. While developing the project, I learned NoSQL database management with MongoDB and built the communication between React and .NET through a REST API. My friend Burak also tested the project and found it quite useful.

Event Management System

This project was one of the tasks assigned to me during an online internship program. I developed an event management system with core features like creating, editing, deleting events, and managing participants. After successfully completing the internship task, I didn't want to abandon the project.

I wanted to continue developing this project for Kumluca, the town where I live. I aim to create a platform for festivals, concerts, local events, and meetings held in Kumluca. Building a system that makes events in our district more organized and accessible is a valuable goal for me both technically and socially. Being able to contribute to Kumluca in this way would make me very happy.
